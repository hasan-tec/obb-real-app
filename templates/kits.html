What is Age Rank?

Age Rank controls FIFO (oldest ships first). Lower number = older batch = ships first. The system reads the SKU prefix (letters before the first digit) and auto-computes the rank — so you almost never need to touch it manually.

SKU Prefix → Rank Formula
A=1   B=2   ...   Z=26
AA=27   AB=28   ...   AZ=52
BA=53   BB=54   ...   BT=72   BV=74
CA=79   CI=87   CJ=88   CK=89
WK=10001   WKA=10002   ...   WKH=10009
SKU digits after prefix = trimester + size (e.g. CJ21 = T2, Universal)
When Adding a Kit
  • Leave Age Rank at 0 → system auto-calculates from SKU. Best practice.
  • Enter any other number → that value is used and 🔒 locked as MANUAL. Recalculate button will skip it.
When Editing a Kit
  • Change SKU only → rank auto-recomputes from new SKU ✅
  • Change SKU + type a custom rank → your rank is kept, 🔒 locked as MANUAL
  • Same SKU, set rank to 0 → resets to auto from SKU ✅
  • Same SKU, type different rank → 🔒 locked as MANUAL
↻ Recalculate Button — When to Click
  • After a bulk import of new kits to set all their ranks at once.
  • If you suspect some ranks are wrong or out of sync with SKUs.
  • SAFE to click any time — it never touches kits marked 🔒 MANUAL.
  • You do not need to click it after every single edit — only AUTO kits get updated.
AUTO vs 🔒 MANUAL
AUTO  = system computed from SKU. Recalculate can update it.

🔒 MANUAL  = you set it. Recalculate will not change it. To reset to auto, edit the kit and set rank to 0.
